# Text_Rain
CSCI 4611 assignment 1: Text Rain

Reimplement text rain by Romy Achituv and Camille Utterback: http://camilleutterback.com/projects/text-rain/ using Processing. You could use the pre-recorded video or read live video from a webcam attached to your computer as the input. The text characters will fall like rain until they bump into a dark object at which point they should stop falling. Pressing space bar will enter debug mode.
